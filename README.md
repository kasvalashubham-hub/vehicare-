A personalized vehicle maintenance scheduler tailored for Indian road conditions.

## Features
- **Vehicle Conditioning**: Adjusts schedules based on traffic, rough roads, and highway usage.
- **Maintenance Score**: Calculates a health score for your vehicle.
- **Critical & Upcoming**: Highlights immediate formatting needs vs long-term service.
- **Upgrades**: Suggests accessories relevant to your driving style (e.g., Dashcams for city, skid plates for rough roads).

## Deployment

### Deploy with Vercel

2.  **Install Vercel CLI**:
    ```bash
    npm i -g vercel
    ```
3.  **Deploy**:
    Run the following command in this directory:
    ```bash
    vercel
    ```
    Follow the prompts to deploy.

### Manual Deployment
You can also drag and drop this folder into your [Vercel Dashboard](https://vercel.com/dashboard).
